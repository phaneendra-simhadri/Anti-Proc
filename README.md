# APEX (Anti-Procrastination Execution System)

A strict execution web app for computer science students.

## What it enforces

- Exactly 3 mandatory tasks per day.
- No carry forward: unfinished tasks count as a missed day.
- Streak resets on missed day.
- XP gains on complete days, XP deduction and penalty on missed days.
- Daily consistency heatmap with weekly/monthly summaries.
- Deep work session logging (60-90 minute planned blocks).
- Weekly reflection prompt every 7 logged days.

## Run

Open `index.html` directly in a browser.

## Data

All data is stored locally in browser `localStorage` under key `apex-v1`.
